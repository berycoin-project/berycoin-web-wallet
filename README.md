# berycoin-web-wallet
A web wallet for BERYCOIN

nvm
```
curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.11/install.sh | bash
source ~/.bashrc
nvm i v6
nvm use v6
```

setup
```
git clone https://github.com/berycoin-project/berycoin-web-wallet
cd berycoin-web-wallet
npm install
npm start
```

host
```
http://127.0.0.1:8080
````
